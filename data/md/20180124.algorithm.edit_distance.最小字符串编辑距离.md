# 最小字符串编辑距离

###### 2018.01.24

## 问题描述

    给定两个字符串m和n，只允许进行如下三种操作：
    插入，例如：ab -> abc
    删除，例如：abc -> ab
    替换，例如：abc -> abd
    那么请求出将m变成n的最小操作次数（也就是最小编辑距离）

## 解法

```
if len(m) = 0
    d[j][k] = len(n)
elseif  len(n) = 0
    d[j][k] = len(m)
else
    if m[j] == n[k] 
        d[j][k] = d[j - 1][k - 1]
    else
        d[j][k] = min(d[j - 1][k] + 1, d[j][k - 1] + 1, d[j - 1][k - 1] + 1)
```
